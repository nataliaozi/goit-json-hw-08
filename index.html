Moduł 8 Zadanie Domowe: JSON i JavaScript

1. Analiza pliku JSON

Oto dane z pliku user.json:

        {
  "name": "Josh",
  "weight": 175,
  "age": 30,
  "eyecolor": "brown",
  "isHappy": true,
  "cars": ["Chevy", "Honda"],
  "favouriteBook": {
    "title": "The Last Kingdom",
    "author": "Bernard Cornwell",
    "rating": 8.38
  }
}

Pary klucz - wartość, oraz określenie typu danych

  • (klucz)name - (wartość)Josh , jest to string (ciąg znaków)
  • (klucz)weight - (wartość)175, jest to number (liczba)
  • (klucz)age - (wartość) 30, jest to number (liczba)
  • (klucz)eyecolor - (wartość)brown, jest to string (ciąg znaków)
  • (klucz) isHappy - (wartość)true, jest to boolean (wartość logiczna)
  • (klucz)cars - (wartość) 
    ["Chevy", "Honda"]
    , jest to table (tablica z tekstami)
  • (klucz) favouriteBook - (wartość) 
    {
    "title": "The Last Kingdom",
    "author": "Bernard Cornwell",
    "rating": 8.38
  }
    , jest to object (obiekt)

Wyjaśnienie struktury JSON

  • JSON jest formatem, który służy do zapisywania danych w formie par klucz - wartość.
  • Wartości mogą być różnych typów, takich jak: tekst(string), liczba (number), wartość logiczna (boolean), tablica (table), czy też obiekt (object).
  • Klucze w JSON są zawsze ciągami w podwójnych cudzysłowach.
